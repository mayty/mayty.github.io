ПРАВИЛА ПРОВЕДЕНИЯ ЗАЧЕТА

 

Зачет проводится в электронном виде с использованием специально разработанного на кафедре программного обеспечения.

Предусмотрены вопросы двух видов:
1. MCSA (Multiple Choice Single Answer) -- нужно выбрать один правильный вариант из нескольких предложенных (по схеме один из пяти).
2. MCMA (Multiple Choice Multiple Answer) -- нужно выбрать несколько правильных вариантов из всех предложенных (по схеме два из шести) -- учитывается каждый правильный подответ.

На зачете задаются по два вопроса (MCSA и MCMA) из каждого рассмотренного на лекциях раздела.

За правильный овет на вопрос MCSA дается два балла. За правильный подответ на вопрос MCMA дается один балл.

Для получения зачета необходимо набрать половину от возможного количества баллов, с учетом статистической вероятности угадывания.

Пример вопроса MCSA:

Что изображено на рисунке?
( ) Коммутатор
( ) Медиаконвертер
( ) Модуль маршрутизатора
( ) Оконечный концентратор
( ) Ни один из перечисленных вариантов

Пример вопроса MCMA:

Какие существуют базовые модели QoS? (Выберите два варианта)
[ ] Best Effort
[ ] Best Match
[ ] Integrated Services
[ ] Link State
[ ] Network Load Balancing
[ ] SMART
