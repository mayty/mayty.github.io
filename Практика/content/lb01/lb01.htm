ЛАБОРАТОРНАЯ РАБОТА №1

 

Цель: получить практические навыки настройки управляемого коммутатора.

Лабораторная работа выполняется студентами индивидуально (на базе эмулятора КС, по варианту) или коллективно (на базе реального оборудования, каждый студент настраивает свое оборудование в классе).

Задание на лабораторную работу может содержать следующие пункты:

1. Сбросить текущие настройки коммутатора, то есть установить заводские настройки.

2. Научиться использовать web-интерфейс для аминистрирования. Для этого подключить к коммутатору пользовательскую станцию. Помнить об IP-адресе и учетной записи по умолчанию.

3. Научиться использовать консоль. Для этого пользовательскую станцию подключить посредством последовательного интерфейса. Помнить о параметрах RS-232 по умолчанию.

4. Согласно варианту задания назначить коммутатору новые IP-параметры.

5. Согласно варианту задания создать новую учетную запись для удаленного доступа.

6. Научиться принудительно задавать скорость и режим физического соединения портов.

7. Согласно варианту задания внести изменения в таблицу MAC-адресов.

8. Согласно варианту задания настроить Port Security.

9. Согласно варианту задания настроить Port Mirroring.

10. Изучить возможности защиты от «штормов» и включить защиту.

11. Изучить возможности энергосбережения.

12. Изучить возможности диагностики и отслеживания определенных событий.

13. Ознакомиться с оригинальными возможностями коммутатора, если таковые имеются.
